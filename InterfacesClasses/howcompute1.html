How Do You Compute With an Object?

  • To invoke a method of an object, we send the object a message.
  • For example: To invoke the area method of an object obj1, we write 
              obj1.area()
  • If obj1 is an object of class C, that syntax invokes the area method of class C.
(We are using the Java syntax for sending the area message to the object obj1. Some languages use a different syntax. In Racket, for example, we would write (send obj1 area).)
