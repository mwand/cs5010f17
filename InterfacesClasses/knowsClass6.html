obj2:



x = 15

      class C1 {
          int x;
          int y;
          int r;
      
          public int foo () { return x + y; }
          public int bar (int n) { return r + n; }
          public int baz (int n) { return this.foo() + n; }
          ...
      }

Here's another object of a different class C2. If we send a message to obj3, then obj3's methods, as defined in class C2, will be invoked.

obj3:



a = 15
b = 35
c = 5 

          
          class C2 {
              int a;
              int b;
              int c;
          
              public int foo () { return a + b; }
              public int bar (int n) { return c * n; }
              public int baz (int n) { return this.foo() + n; }
              ...
          }

So

obj2.bar(8) = 5 + 8 = 13
obj3.bar(8) = 5 * 8 = 40
