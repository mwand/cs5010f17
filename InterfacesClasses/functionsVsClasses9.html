      // Constructor template for Circle
      //
      //     new Circle (int, int, int, ColorString)
      //
      // Interpretation: a circle on the canvas
      
      class Circle implements Shape {
          int x;            // x-position of center
          int y;            // y-position of center
          int r;            // radius
          ColorString c;    // color of this circle
          Image img;        // image of this circle
      
          // this constructor initializes the fields
      
          Circle (int x, int y, int r, ColorString c) {
              this.x = x;
              this.y = y;
              this.r = r;
              this.c = c;
              this.img = Image.circleImage (r, "solid", c);
          }
      
          // methods
      
          // RETURNS: the weight of this shape
      
          public double weight () {
              return Math.PI * r * r;
          }
      
          // RETURNS: a scene like the given one, but with this
          //     shape painted on it
      
          public Scene addToScene (Scene sc) {
              return Image.placeImage (img, x, y, sc);
          }
      }

The class definition above assumes Scene, ColorString, and Image data types analogous to those we've been using in Racket, and assumes static methods Image.circleImage and Image.placeImage analogous to the circle and place-image functions we've been using in Racket.
